## 搜索拐杖

搜索拐杖（SearchCrutch），基于[Google搜索拐杖](http://www.kkksou.com/product/gc.html)修改，支持manifest 2。

## 实现功能
1, 在Google搜索不通时，显示切换搜索按钮；
2, 可切换 谷歌复原/百度/必应/雅虎/搜狗/有道；
3, 保持搜索关键词，无需重输；
4, 支持快捷切换Http/Https访问Google Reader；
5, 支持Google引用页重试直接访问(新)(?)。
